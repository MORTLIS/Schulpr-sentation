Linux Installation

1. Einleitung

Linux ist ein Betriebssystem. Es ist schnell und kostenlos.

2. Voraussetzungen

  • Ein Computer
  • Ein USB-Stick (8 GB oder mehr)
  • Eine Linux-ISO-Datei

3. Download und Vorbereitung

Lade eine Linux-ISO-Datei herunter. Zum Beispiel Ubuntu.

Nutze ein Programm wie Rufus, um den USB-Stick zu erstellen.

4. BIOS/UEFI-Konfiguration

Bevor Sie von dem USB-Stick booten, müssen Sie die BIOS-Einstellungen ändern:

  • Starten Sie den Computer neu und rufen Sie das BIOS auf (normalerweise mit F2, F12, Esc oder Del beim Start).

Gehen Sie zum Menü Boot:

  • Setzen Sie die USB-Boot-Priorität höher als die Festplatte.
  • Falls Ihr Laufwerk im UEFI-Modus arbeitet, deaktivieren Sie Secure Boot.

Speichern Sie die Änderungen und starten Sie den Computer neu.

5. Installation

Nach dem Booten wählen Sie "Try Linux" (um das System ohne Installation zu testen) oder "Install Linux".

Wählen Sie die Sprache und das Tastaturlayout aus.

Verbinden Sie sich mit dem WLAN (falls nötig).

Wählen Sie eine Installationsoption:

  • Festplatte löschen und Linux installieren – wenn Sie Windows entfernen möchten.
  • Linux neben Windows installieren – falls Sie beide Betriebssysteme nutzen möchten (Dual Boot).
  • Andere Option – für eine manuelle Partitionierung (für erfahrene Nutzer).

Wählen Sie Ihre Zeitzone.

Erstellen Sie einen Benutzernamen und ein Passwort.

Warten Sie, bis die Installation abgeschlossen ist, und starten Sie den Computer neu.

6. Abschluss und erste Schritte

Die Installation ist fertig! Starte den Computer neu.

Willkommen bei Linux!
